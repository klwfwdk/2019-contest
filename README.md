# 2019年前端星计划

这是星计划校招选拔的综合挑战题

## 题目描述

用 JavaScript 实现一个推箱子游戏

![](https://p3.ssl.qhimg.com/t01a01fe1b96d05edcd.png)

## 要求

- 能够选择关卡和难度
- 能够正确判断失败和胜利

## 加分项

- 支持移动设备
- 比较优秀的UI和交互
- 代码实现优雅，注释完整

## 如何提交

参加 2019 年前端星计划的学生 fork 本项目，编辑.75camp文件，添加姓名和学号后4位，用于验证身份。

完成作品后将项目URL提交到奇舞学院测试试卷的相关题目的答案中。

## 已经实现的功能
- 已经实现能够选择关卡难度
- 能够判断成功，能判断简单的失败情况
